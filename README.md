## nonebot_plugin_apexranklookup

一个基于nonebot的插件，用于查询Apex英雄的

> 地图轮换
> 
> 玩家信息
> 
> 复制器
> 
> 猎杀门槛

## api key申请
在https://apexlegendsapi.com/ 获取APIkey填入.env 或者config.py中
`apex_api_token`

## 使用
#### .map/.地图
查看当前大逃杀、竞技场模式轮换地图，以及地图剩余时间。

#### .stat/.查询 origin_id
根据origin_id查询玩家信息（id、等级、段位、当前状态）

#### .crafting/.复制器
查询当前复制器轮换物品。

#### .pd/.猎杀
查询全平台大逃杀、竞技场猎杀最低分和大师段位以上的人数。
